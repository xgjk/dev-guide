### 4.1 创建产品知识

- **接口名称**: 创建产品知识
**基本信息**

| 项目 | 说明 |
|---|---|
| 接口地址 | `/knowledge/createProductKnowledge` |
| 请求方式 | POST |
| Content-Type | application/json |
| 数据量级别 | 小 |
| 预估响应体积 | 约 0.5KB |


**请求体参数**

| 字段      | 类型   | 必填 | 说明         |
| --------- | ------ | ---- | ------------ |
| drugId    | number | 是   | 品种 ID      |
| category  | string | 是   | 分类         |
| title     | string | 是   | 知识标题     |
| content   | string | 是   | 知识正文     |
| sortOrder | number | 否   | 排序，默认 0 |

**请求示例 (cURL)**

```bash
curl -X POST 'https://cwork-web-test.xgjktech.com.cn/open-api/tbs-admin/knowledge/createProductKnowledge' \
  -H 'appKey: YOUR_APP_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
  "drugId": 2001,
  "category": "核心卖点",
  "title": "首剂起效速度",
  "content": "在目标人群中 2 周即可观察到关键指标改善趋势。",
  "sortOrder": 10
}'
```

**请求体示例**

```json
{
  "drugId": 2001,
  "category": "核心卖点",
  "title": "首剂起效速度",
  "content": "在目标人群中 2 周即可观察到关键指标改善趋势。",
  "sortOrder": 10
}
```

**成功响应示例**

```json
{
  "resultCode": 1,
  "resultMsg": "success",
  "data": 5012
}
```

**响应参数**

`data` 类型为 `String` (ID):

| 字段名 | 类型 | AI决策 | 说明 |
|---|---|---|---|
| `data` | String | 是 | 返回的主键 ID |

### 4.2 创建角色画像（医生画像）

- **接口名称**: 创建角色画像
**基本信息**

| 项目 | 说明 |
|---|---|
| 接口地址 | `/rolePersona/createRolePersona` |
| 请求方式 | POST |
| Content-Type | application/json |
| 数据量级别 | 小 |
| 预估响应体积 | 约 0.5KB |


**请求体参数**

| 字段            | 类型    | 必填 | 说明                     |
| --------------- | ------- | ---- | ------------------------ |
| name            | string  | 是   | 画像名称/医生名          |
| surname         | string  | 否   | 姓氏或补充称呼           |
| title           | string  | 否   | 职称                     |
| description     | string  | 否   | 简介                     |
| personaConfig   | string  | 否   | 人设配置                 |
| trustInitial    | number  | 否   | 初始信任值，默认 50      |
| patienceInitial | number  | 否   | 初始耐心值，默认 60      |
| isPreset        | boolean | 否   | 是否预置画像，默认 false |

**请求示例 (cURL)**

```bash
curl -X POST 'https://cwork-web-test.xgjktech.com.cn/open-api/tbs-admin/rolePersona/createRolePersona' \
  -H 'appKey: YOUR_APP_KEY' \
  -H 'Content-Type: application/json' \

  "name": "王主任",
  "surname": "王",
  "title": "心内科主任医师",
  "description": "注重循证，沟通节奏快",

  "trustInitial": 45,
  "patienceInitial": 55,

}'
```

**请求体示例**

```json
{
```

### 4.6 查询角色画像列表

**基本信息**

| 项目 | 说明 |
|---|---|
| 接口地址 | `/rolePersona/listRolePersonas` |
| 请求方式 | POST |
| Content-Type | application/json |
| 数据量级别 | 中 |
| 预估响应体积 | 约 5-10KB |

**请求参数 (JSON Body)**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| - | Object | 否 | 传空对象 `{}` 即可获取全部画像 |

**请求示例 (cURL)**

```bash
curl.exe --compressed --location --request POST "https://cwork-web-test.xgjktech.com.cn/open-api/tbs-admin/rolePersona/listRolePersonas" \
--header "appKey: YOUR_APP_KEY" \
--header "Content-Type: application/json" \
--data "{}"
```

**响应参数**

| 字段名 | 类型 | AI决策 | 说明 |
|---|---|---|---|
| resultCode | Integer | 是 | 1=成功 |
| data | Array | 是 | 画像列表 |
| └ id | Long/String | 是 | 画像 ID |
| └ name | String | 是 | 画像名称 |
| └ surname | String | 否 | 姓氏/称呼 |
| └ title | String | 否 | 职称 |
| └ description | String | 否 | 画像简介 |
| └ personaConfig | String | 否 | 人设详细配置（包含性格特征、沟通风格、行为倾向等 Markdown 内容） |
| └ trustInitial | Integer | 否 | 初始信任值（0-100） |
| └ patienceInitial | Integer | 否 | 初始耐心值（0-100） |
| └ isPreset | Boolean | 是 | 是否为系统预置画像 |
