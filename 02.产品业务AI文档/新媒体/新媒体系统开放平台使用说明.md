## 新媒体系统开放平台 Open API 接口文档（KOL 相关）

### 一、概述

本文档用于说明「新媒体系统开放平台」开放接口能力（以 KOL 相关接口为主）。

系统支持两种认证方式（二选一）：

- **Bearer Token**：`Authorization: Bearer {token}`
- **AppKey**：`appKey: {你的AppKey}`

---

### 二、通用说明

#### 2.1 访问地址

```text
http://{域名}/{接口地址}
```

示例：`http://hk-al-xg-node.mediportal.com.cn/api/kol/getposts`

#### 2.2 环境信息

| 环境     | 域名 |
| -------- | ---- |
| 生产环境 | `http://hk-al-xg-node.mediportal.com.cn` |

#### 2.3 公共请求头

| Header | 说明 | 是否必填 |
| ------ | ---- | -------- |
| `appKey` | 应用密钥（联系管理员获取） | 是（推荐方式） |
| `Authorization` | Bearer Token（如已登录系统） | 否 |
