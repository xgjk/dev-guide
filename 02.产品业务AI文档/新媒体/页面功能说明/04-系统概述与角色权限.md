# 数擎罗盘 — 系统概述与角色权限

## 1. 系统简介

**系统名称**：数擎罗盘（内部代号 VueKOL / kol_cms_system）

**系统定位**：以数据为引擎的智能化数据分析平台

**核心业务**：
- KOL 社交媒体视频监控 — 追踪抖音、小红书、视频号等平台的 KOL（达人/医生）账号动态
- 多平台销售数据管理 — 汇总淘宝、京东、美团、饿了么、抖音、拼多多、京东到家等 7 大电商平台的销售数据

**生产地址**：https://hk-al-xg-node.mediportal.com.cn

---

## 2. 登录方式

### 2.1 钉钉 OAuth 登录（推荐）
- 在钉钉内打开系统时，支持一键授权登录
- 系统会自动检测是否在钉钉内置浏览器中，如果不是会提示跳转到系统浏览器
- 登录后自动获取用户的钉钉信息（姓名、头像等）

### 2.2 API Key 登录（备用）
- 输入专属 API Key 完成登录
- 适合非钉钉环境下的访问

### 2.3 快速登录
- 系统会记住上次登录的用户信息，下次打开时免重复输入

## 3. 用户角色与权限

系统采用基于角色的权限控制，不同角色能看到不同功能菜单。

### 3.1 超级管理员（role=9）
- 可访问**全部功能菜单**
- 额外拥有系统管理模块：用户管理、权限配置、品牌管理、预警规则管理
- 可查看用户操作日志

### 3.2 普通用户
- 根据管理员分配的权限，看到对应的菜单
- 权限由管理员在"权限配置"页面统一管理

### 3.3 默认用户（无权限信息）
- 仅能看到三个基础页面：首页、KOL列表、视频列表

### 3.4 权限刷新
- 首页右上角有"刷新权限"按钮
- 当管理员修改了你的权限后，点击此按钮即可立即生效，无需重新登录

---

## 4. 页面布局

### 4.1 左侧边栏
- 系统的主要导航菜单
- 菜单项根据用户权限动态显示/隐藏
- 菜单分组：KOL监控、销售数据、日志、系统管理等

### 4.2 顶部标签页
- 类似浏览器的多标签页导航
- 每打开一个功能页面会新增一个标签
- 首页标签固定不可关闭
- 可在不同标签间快速切换

### 4.3 页面水印
- 所有页面都带有基于用户名生成的水印
- 水印内容：用户名末字 + 用户ID末3位
- 作用：防止截图泄露，便于追溯信息来源

## 6. 自动刷新机制

系统各模块的数据会自动定时更新，无需手动刷新：

| 数据类型 | 刷新频率 | 说明 |
|----------|----------|------|
| KOL 用户信息 | 每 1 小时 | 页面显示下次更新倒计时 |
| 视频列表 | 每 1 小时 | 自动获取最新发布的视频 |
| 视频监控 | 每 20 分钟 | 重点监控的视频更频繁更新 |
| 首页提醒动态 | 每 5 分钟 | 自动滚动展示最新提醒 |
| 首页统计数据 | 每 5 分钟 | 监控数、视频总数等 |

---

## 7. 通知推送

- 系统通过**钉钉消息**推送重要提醒
- 提醒类型：KOL 新作品提醒、视频监控提醒
- 推送订阅：管理员可在用户管理页面为用户配置推送订阅
- 用户可查看提醒历史记录（首页"最新提醒动态"区域）
