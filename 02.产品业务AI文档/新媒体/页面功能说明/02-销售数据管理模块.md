# 销售数据管理模块

本模块是系统最庞大的功能模块，涵盖 GMV 看板、业绩 ROI、品牌销售、费用管理、目标管理等 11 个子页面，实现从数据同步到分析展示的全链路管理。

---

## 1. GMV 看板（/sales/gmv-dashboard）

### 1.1 功能说明
年度全网 GMV（Gross Merchandise Volume，商品交易总额）达成情况总览，是管理层最关注的核心看板。

### 1.2 时间进度行
页面顶部展示三个维度的时间进度条：
- **年度时间进度**：当前日期在全年中的位置（如：已过 42%）
- **季度时间进度**：当前日期在本季度中的位置
- **月度时间进度**：当前日期在本月中的位置

进度条直观显示"时间消耗比例"，便于与"目标达成比例"对比。

### 1.3 品牌维度达成表格
- 每行一个品牌，展示年度目标 vs 实际达成
- 可按类型筛选：OTC（非处方药）、保健品等
- 支持品牌颜色标识，便于视觉区分

### 1.4 月份切换
- 选择特定月份查看该月的漏斗分析
- 选择"实时统计"查看截至当前的累计数据

### 1.5 M2/M3 目标切换
- **M3 目标**：年初制定的初始目标
- **M2 目标**：年中调整后的目标
- 可在两个目标之间切换查看达成情况

### 1.6 日期筛选
- 可自定义截止日期，查看截至指定日期的达成情况

## 2. 业绩 ROI（/sales/target-stats）

### 2.1 功能说明
查看各分组的年度业绩与销售数据，分析品牌投入产出比（ROI）。

### 2.2 品牌 ROI 卡片
- 每个品牌一个卡片，展示 ROI 数据
- 可进行 ROI 对比分析，跨品牌比较

### 2.3 业绩概览
- 各分组的年度业绩汇总
- 月/年销售达成统计

### 2.4 趋势分析
- 12 个月销售趋势折线图
- 直观展示全年销售走势

---

## 3. 品牌销售（/sales/brand-stats）

### 3.1 功能说明
统一管理多平台销售数据，支持品牌维度的数据查看和跨平台对比。

### 3.2 品牌切换
- 选择不同品牌查看其销售数据
- 品牌列表从系统品牌管理中获取

### 3.3 品牌销售占比饼图
- 各品牌在总销售中的占比可视化
- 直观展示品牌贡献度

### 3.4 平台销售详情
- 每个平台的具体销售数据表格
- 支持跨平台对比分析

---

## 4. 分组统计（/sales/group-stats）

### 4.1 功能说明
查看分组员工负责的监控视频发布后 14 天内的增量数据，评估运营效果。

### 4.2 数据指标
- **总数**：视频相关数据的汇总
- **互动数**：点赞、评论、转发等互动数据
- **花费数**：投放花费数据
- **增量趋势**：数据随时间的变化趋势

### 4.3 筛选功能
- 按分组筛选
- 按用户筛选

### 4.4 概览卡片
- 汇总卡片展示关键指标

---

## 5. 费用支出（/fee/view）

### 5.1 功能说明
各品牌费用数据的统一管理，支持录入、导入和审批。

### 5.2 筛选维度
- 年份
- 月份
- 平台
- 品牌

### 5.3 费用录入
- **手动新增**：逐条录入费用数据
- **Excel 导入**：批量导入费用数据
  - 支持下载导入模板
  - 导入前可预览数据
- **免费分发费用**：记录免费分发的相关费用

### 5.4 费用状态管理
- 费用数据有状态流转（如：待审批、已审批、已驳回等）
- 不同状态用不同颜色标识

---

## 6. 销售日报（/sales/daily）

### 6.1 功能说明
每日销售数据的录入和统计，支持多平台数据管理。

### 6.2 支持平台
淘宝、京东、美团、饿了么、抖音、拼多多、京东到家

### 6.3 快速日期选择
- 今天
- 本周
- 本月
- 自定义日期范围

### 6.4 数据录入
- **手动新增**：逐条录入每日销量
- **Excel 导入**：批量导入销售日报数据

---

## 7. 目标管理（/sales/target-manage）

### 7.1 功能说明
设定和管理品种/品牌的年度销售目标，是 GMV 看板的数据基础。

### 7.2 目标设定
- 选择品种（品牌）
- 选择年份
- 选择目标阶段：M1/M2/M3

### 7.3 行内编辑
- 点击表格中的数值可直接修改
- 无需弹窗，操作便捷
- 修改后自动保存

### 7.4 指标管理
- 点击"指标管理"按钮管理所有指标项
- 支持：新增、编辑、删除、排序
- 每个指标可添加备注说明

### 7.5 手动计总
- 点击"手动计总"按钮触发数据汇总
- 自动计算各月数据的合计值
- 计总范围会显示在按钮上（如：1~12月）

### 7.6 目标阶段说明
| 阶段 | 说明 |
|------|------|
| M3 | 年初制定的初始目标 |
| M2 | 年中调整后的目标 |
| M1 | 最终确定的目标 |

## 9. 数据同步（/sales/raw-data）

### 9.1 功能说明
查看各平台同步过来的原始数据，并可手动触发数据同步。

### 9.2 三种同步操作

| 同步类型 | 覆盖平台 | 说明 |
|----------|----------|------|
| 同步主平台 | 美团、阿里、京东、抖音、京东到家 | 同步销售主数据 |
| 同步下单盒数 | 阿里、京东、抖音 | 同步下单数量数据 |
| 同步数擎 | 拼多多 | 同步拼多多平台数据 |

### 9.3 数据流向
```
各平台 API 接口
    ↓ 同步操作
原始同步数据表（/sales/raw-data 可查看）
    ↓ 汇总处理
中转汇总表（/sales/gmv-summary 可查看）
    ↓ 展示
GMV 看板 / 数据总览
```

## 10. 中转表数据（/sales/gmv-summary）

### 10.1 功能说明
查看 GMV 中转汇总数据，是原始数据到看板数据的中间层。

### 10.2 筛选功能
- 平台
- 品牌
- 日期范围

### 10.3 平台类型区分
- **B2C 平台**：淘宝、京东、抖音、拼多多等（直接面向消费者）
- **O2O 平台**：美团、饿了么、京东到家等（线上到线下）

---

## 11. 平台统计（/sales/shop-stats）

### 11.1 功能说明
多平台销售数据的跨平台对比分析。

### 11.2 销售占比饼图
- 最近 30 天各平台销售占比
- 点击平台可下钻至品牌维度

### 11.3 品牌占比图
- 各品牌在总销售中的占比

### 11.4 品牌详情表格
- 品牌的详细销售数据
- 可按平台筛选

## 13. 销售模块页面关系

| 页面 | 主要用途 | 数据来源 |
|------|----------|----------|
| GMV 看板 | 年度达成总览 | 中转汇总表 |
| 业绩 ROI | 分组业绩分析 | 销售数据+费用数据 |
| 品牌销售 | 品牌维度分析 | 中转汇总表 |
| 分组统计 | 运营效果评估 | 视频监控数据 |
| 费用支出 | 费用管理 | 手动录入/Excel导入 |
| 销售日报 | 每日数据录入 | 手动录入/Excel导入 |
| 目标管理 | 目标设定 | 手动录入 |
| 数据总览 | 全数据查看 | 中转汇总表 |
| 原始同步数据 | 数据同步 | 平台 API |
| 中转表数据 | 中间层数据 | 原始数据汇总 |
| 平台统计 | 跨平台对比 | 中转汇总表 |
