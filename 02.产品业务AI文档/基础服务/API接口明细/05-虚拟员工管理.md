## 一、能力概述

| 接口 | 说明 |
|------|------|
| 5.1 我的虚拟助理列表 | 获取当前账号关联的所有虚拟员工（NPC） |
| 5.2 添加虚拟员工 | 创建并绑定一个新的虚拟员工角色 |
| 5.3 修改虚拟员工信息 | 更新名称、备注等基础信息 |
| 5.4 删除虚拟员工 | 解除绑定并逻辑删除实体 |

---

## 二、典型业务流程

### 场景：维护个人专属 AI 助理

1. 调用 **5.2 添加虚拟员工**（`POST /cwork-user/virtual-employee/add`）创建一个名为“知识助手”的角色。
2. 得到 `virtualEmpId`。
3. 调用 **5.1 我的虚拟助理列表**（`GET /cwork-user/virtual-employee/list`）进行确认。

---

## 三、接口详细说明

### 5.1 我的虚拟助理列表

获取当前登录用户或其指定可见范围内的虚拟员工列表。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/cwork-user/virtual-employee/list` |
| 请求方式 | `GET` |
| 数据量级别 | 小（通常 ≤ 10 条） |

**请求参数**

无。

**响应参数**

`data` 类型为 `List<VirtualEmployeeVO>`，详见 **[4.1](#41-virtualemployeevo)**。

**请求示例**

```bash
curl -X GET 'https://{域名}/open-api/cwork-user/virtual-employee/list' \
  -H 'appKey: YOUR_API_KEY'
```

---

### 5.2 添加虚拟员工

由当前真实用户发起，创建一个逻辑绑定的虚拟员工实体。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/cwork-user/virtual-employee/add` |
| 请求方式 | `POST` |
| Content-Type | `application/json` |
| 数据量级别 | 小 |

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `name` | String | 是 | 虚拟员工姓名 |
| `remark` | String | 否 | 备注信息 |

**响应参数**

`data` 类型为 `Long`，返回新增成功的私有 `virtualEmpId`。

---

### 5.3 修改虚拟员工信息

更新指定虚拟员工的基础属性。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/cwork-user/virtual-employee/update` |
| 请求方式 | `POST` |
| Content-Type | `application/json` |
| 数据量级别 | 小 |

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `virtualEmpId` | Long | 是 | 虚拟员工 ID |
| `name` | String | 否 | 新的姓名 |
| `remark` | String | 否 | 新的备注 |

**响应参数**

`data` 类型为 `Boolean`。

---

### 5.4 删除虚拟员工

解除人员绑定并逻辑注销虚拟实体。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/cwork-user/virtual-employee/delete` |
| 请求方式 | `POST` |
| 数据量级别 | 小 |

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `virtualEmpId` | Long | 是 | 待删除的虚拟员工 ID |

**响应参数**

`data` 类型为 `Boolean`。

---

## 四、公共数据结构

### 4.1 VirtualEmployeeVO

| 字段 | 类型 | AI决策 | 说明 |
| --- | --- | ----- | --- |
| `virtualEmpId` | Long | 是 | 虚拟员工主键 ID |
| `name` | String | 是 | 虚拟员工名称 |
| `remark` | String | 否 | 备注 |
| `ownerName` | String | 否 | 所属真实员工姓名 |
