## 一、能力概述

| 接口 | 说明 |
|------|------|
| 4.1 我的个人标签列表 | 获取当前登录用户定义的全部标签 |
| 4.2 创建个人标签 | 新增自定义标签名称 |
| 4.3 更新个人标签 | 修改已有标签的名称 |
| 4.4 删除个人标签 | 物理删除指定的标签定义 |

---

## 二、典型业务流程

### 场景：个人工作分类管理

1. 调用 **4.1 我的个人标签列表**（`GET /work-report/label/listLabel`）获取现有标签及其 `id`。
2. 若需新分类，使用 **4.2 创建个人标签**（`POST /work-report/label/createLabel`）新增，例如“重要项目”。
3. 得到 `id` 后，可在汇报等业务模块中进行绑定（详见具体业务 API）。

---

## 三、接口详细说明

### 4.1 我的个人标签列表

获取当前用户创建的所有个人标签集合。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/work-report/label/listLabel` |
| 请求方式 | `GET` |
| 数据量级别 | 小（通常 ≤ 50 条） |

**请求参数**

无。

**响应参数**

`data` 类型为 `List<ReportLabelVO>`，详见 **[4.1](#41-reportlabelvo)**。

**请求示例**

```bash
curl -X GET 'https://{域名}/open-api/work-report/label/listLabel' \
  -H 'appKey: YOUR_API_KEY'
```

---

### 4.2 创建个人标签

新增一个自定义个人标签名称。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/work-report/label/createLabel` |
| 请求方式 | `POST` |
| Content-Type | `application/json` |
| 数据量级别 | 小 |

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `name` | String | 是 | 标签名称 |

**响应参数**

`data` 类型为 `ReportLabelVO`，返回生成成功的标签对象（含 `id`）。

---

### 4.3 更新个人标签

修改已有个人标签的文字显示。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/work-report/label/updateLabel` |
| 请求方式 | `POST` |
| Content-Type | `application/json` |
| 数据量级别 | 小 |

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `id` | Long | 是 | 标签 ID |
| `name` | String | 是 | 新的标签名称 |

**响应参数**

`data` 类型为 `Boolean`。

---

### 4.4 删除个人标签

直接物理删除标签定义。

**基本信息**

| 项目 | 说明 |
| ------------ | ----------------------------------- |
| 接口地址 | `/work-report/label/delLabel` |
| 请求方式 | `GET` |
| 数据量级别 | 小 |

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `id` | Long | 是 | 待删除的标签 ID |

**响应参数**

`data` 类型为 `Boolean`。

---

## 四、公共数据结构

### 4.1 ReportLabelVO

| 字段 | 类型 | AI决策 | 说明 |
| --- | --- | ----- | --- |
| `id` | Long | 是 | 标签主键 ID |
| `name` | String | 是 | 标签名称 |
| `empId` | Long | 是 | 所属用户 ID |
| `corpId` | Long | 否 | 企业 ID |
| `createTime` | Timestamp | 否 | 创建时间 |
